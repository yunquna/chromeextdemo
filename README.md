# 数字叉车 (Digital Forklift) 📦⚡️

> **美西仓储物流人的第一个 Chrome 插件 AI 制作指南**

本项目的**最终目标**是：**授人以渔**。彻底教会即使没有任何编程基础的仓储物流（3PL/WMS）办公人员，依靠 AI (如 ChatGPT / Claude / Gemini) 开发出简单的 Chrome 浏览器插件，快速解决各自手头繁重、重复的网页操作问题。

在物流日常办公中，我们经常面临以下“负载”：
- **手动采集**：在 UPS / FedEx 官网逐个复制单号，效率极低。
- **录入误差**：WMS 库存与账单列表手动誊写，极易出错。
- **格式地狱**：网页数据直接框选粘贴到 Excel 中格式混乱，难以复用。

**“数字叉车”** 的核心理念就是：**像操作叉车搬运实体托盘一样，教你轻松打造自己的虚拟叉车（插件），让自动化为你搬运网页数据。**
